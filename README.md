Pixels
======

This is a small retro game in Flex using Flixel.

* [http://flixel.org](http://flixel.org]

Try
---

Try it online [here](http://sroccaserra.github.com/pixels.html), or download and open the html page in the build directory.

Build
-----

    rake

Requires Rake.

Run Tests
---------

    rake test
